# Quad-Tree
Quad -Tree for Game Terrain with DX11, VS2015 C++ (Optimize Quad Tree Traverse)
- DirectX11
- Quad-Tree
- Tessellation
- Seamless (using Tessellation)
- ImGui
- Game Programming Gems2, 4.5
    - Direct Access QuadTree Lookup
- MemoryPool (using boost::pool)
